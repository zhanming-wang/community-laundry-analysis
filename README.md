# Community Laundry Analysis

Real-time laundry room monitoring with machine learning anomaly detection and smart time recommendations.

**[View Live Dashboard →](https://zhanming-wang.github.io/community-laundry-analysis/)**

---

## Features

- **Live status** — current washer and dryer availability
- **Best time recommendation** — optimal laundry window accounting for wash + dry cycle
- **Peak hour heatmap** — historical busy vs quiet periods by hour
- **Machine health panel** — flags equipment behaving unusually using Isolation Forest anomaly detection
